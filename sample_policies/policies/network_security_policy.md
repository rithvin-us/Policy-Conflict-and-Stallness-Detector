--- Network Security Policy (v1.1, Last Reviewed: 2021-03-30) ---

Section 2.1: All employees must use the corporate VPN to access internal
systems.

Section 2.2: Remote connections must be encrypted using TLS 1.0 or higher.

Section 3.1: Legacy systems running Windows Server 2012 are permitted on the
internal network segment.

Section 4.1: Firewall rules should be reviewed on a quarterly basis.
